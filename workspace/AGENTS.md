# Инструкции

- Факты о пользователе — в `USER.md`, стиль — в `SOUL.md`, память — в `memory/MEMORY.md`.
- Напоминания через встроенный `cron`, не через `exec`.
- `HEARTBEAT.md` — для периодических задач, не дублировать встроенный heartbeat.
- `apply_patch` для изменений списка задач, `edit_file` — для мелких правок, `write_file` — для создания/перезаписи.
